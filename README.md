# GARAGEZKA - Backend Service & API

Repository ini berisi layanan Backend, konfigurasi database, API endpoint, serta dokumentasi teknis untuk aplikasi **GARAGEZKA**.

## Struktur Repositori
- `api/` - Logika pemrosesan endpoint API JSON.
- `config/` - Konfigurasi database dan sistem.
- `database/` - Skema database SQL (`garagezka.sql`).
- `DOKUMENTASI_API_BACKEND.md` - Dokumentasi API dalam format Markdown.
- `DOKUMENTASI_API_BACKEND.html` - Dokumentasi API interaktif HTML.
- `DOKUMENTASI_API_BACKEND_WORD.html` - Dokumentasi API yang diformat khusus untuk dikonversi ke Microsoft Word.

## Fitur Utama Backend
1. **Autentikasi Aman:** Registrasi, Login, Manajemen Sesi, dan Lupa Password.
2. **Manajemen Kendaraan:** CRUD data kendaraan milik pelanggan.
3. **Pemesanan Servis (Booking):** Sistem pendaftaran jadwal servis motor terintegrasi dengan ketersediaan slot waktu.
4. **Notifikasi Sistem:** Notifikasi digital real-time terkait status pemesanan atau status kendaraan pelanggan.
5. **Panel Admin:** Endpoint pengelolaan antrean servis, data pelanggan, pesan kontak, dan manajemen layanan bengkel.
